# ISO 27001:2022 SoA Generator

Single-file ISO 27001:2022 Statement of Applicability (SoA) generator. Search controls, capture applicability and implementation status, and export to JSON or XLSX.

## Features
- Search ISO 27001:2022 controls by ID, title, or description
- Capture applicability, implementation status, exclusions, and inclusion reasons
- Auto-expand incomplete controls and collapse completed ones on load
- Export to JSON or XLSX (multi-sheet)
- Data stored locally in your browser (localStorage)

## Export Sheets (XLSX)
1. Control
2. History
3. Key
4. SOA

## Usage
- Open `index.html` in a browser.
- Enter control decisions and notes.
- Export via **Export XLSX** or **Export JSON**.

## Privacy
All data stays in your browser unless you export it.
